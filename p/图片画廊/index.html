使用 Markdown 创建美丽的交互式图片画廊

Aug 26, 2023
阅读时长: 1 分钟

Hugo Theme Stack 支持使用 Markdown 创建交互式图片画廊。它由 PhotoSwipe 提供支持，其语法灵感来自 Typlog。

要使用此功能，图片必须与 Markdown 文件位于同一目录中，因为它使用 Hugo 的页面束功能来读取图片的尺寸。不支持外部图片。
